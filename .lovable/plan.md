## Problem

The static `public/sitemap.xml` (1,267 lines) is severely outdated compared to the dynamic sitemap edge function. Key gaps:

- **No hreflang annotations** -- missing all `xhtml:link` tags for EN/MS/ZH cross-referencing
- **No PAA articles** -- all 30 PAA article URLs are missing
- **Missing blog articles** -- only 15 of 22 blog slugs present; missing 7 newer articles (emergency-pet-care-guide, pet-nutrition-guide, pet-insurance-comparison, common-pet-illnesses, grooming-tips, vaccinations-schedule, pet-choking-emergency)
- **Old lastmod** -- still shows `2026-01-06` instead of `2026-02-08`
- **Some slug mismatches** -- e.g., `melaka-city` vs `melaka`, `pet-choking-first-aid-malaysia` vs `pet-choking-emergency-malaysia`, `pet-accidents-malaysia` vs `pet-accident-emergency-malaysia`
- **Missing pages** -- no `/ms/` or `/zh/` prefixed URLs at all

## Solution

Replace the entire `public/sitemap.xml` with the output generated by the dynamic sitemap edge function. This is the single authoritative source and already contains:

- All ~474 URLs across 3 languages
- Proper `xhtml:link` hreflang annotations on every entry
- All 22 blog articles, 30 PAA articles, 14 Q&A state pages, 105+ city pages
- Legal pages (English-only)
- Correct lastmod of `2026-02-08`
- Correct priorities (hub cities at 0.9, sub-cities at 0.7, etc.)

## Implementation Steps

1. **Call the sitemap edge function** to get the current XML output
2. **Replace `public/sitemap.xml`** entirely with that output
3. **Verify** the new file includes the `xmlns:xhtml` namespace declaration and all expected URL blocks

## Technical Details

- File: `public/sitemap.xml` (full rewrite)
- Source of truth: `supabase/functions/sitemap/index.ts`
- The static file serves as a fallback when the edge function is not reachable, so it must stay in sync
